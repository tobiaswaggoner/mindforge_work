## Säule 1: Earned Media (PR)

### Warum PR der stärkste Hebel ist

**Die Story verkauft sich selbst:**

> "51-jähriger Thüringer Entwickler baut KI-App, die Schüler vor Klausur-Panik rettet - bootstrapped, ohne Investoren, mit seinen Söhnen als Tester."

**Warum Medien das lieben:**
- **Lokal-Held:** Thüringen, nicht Berlin
- **Aktuell:** KI in Bildung ist DAS Thema
- **Underdog:** Solo-Gründer vs. Big Tech
- **Feel-Good:** Hilft Schülern in Not
- **Authentisch:** Kein VC-Pitch, echtes Problem

## Säule 2: Multiplikatoren (Lehrer)

### Die Mathematik

```
1 Lehrer = 3-5 Klassen = 100-150 Schüler
10 Lehrer = 1.000-1.500 Schüler
50 Lehrer = 5.000-7.500 Schüler

Bei 13% Conversion = 650-975 Premium-User
```

### Lehrer-Strategie

**Phase 1: Pilot-Lehrer (KW 1-4)**

| Aktion | Ziel |
|--------|------|
| 5-10 Lehrer aus Netzwerk ansprechen | Erste Tester |
| Premium-Account kostenlos (für immer) | Incentive |
| Persönlicher Kontakt, kein Massen-Mail | Vertrauen |

**Phase 2: Feedback & Iteration (KW 5-8)**

| Aktion | Ziel |
|--------|------|
| Wöchentliches Feedback einholen | Produkt verbessern |
| Erfolgsgeschichten dokumentieren | PR-Material |
| Probleme sofort fixen | Vertrauen halten |

**Phase 3: Empfehlungs-Kaskade (KW 9+)**

| Aktion | Ziel |
|--------|------|
| Pilot-Lehrer bitten, Kollegen zu empfehlen | Organisches Wachstum |
| Lehrer-Referral-Programm | "Empfehle einen Kollegen" |
| Schulkonferenzen als Multiplikator | Ganze Schule auf einmal |

### Das Lehrer-Angebot

```
FÜR LEHRER:
├─ Premium-Account kostenlos (für immer)
├─ Klassen-Dashboard (Fortschritt sehen)
├─ Direkter Draht zum Gründer (Feedback)
└─ Einfluss auf Feature-Entwicklung

IM GEGENZUG:
├─ Ehrliches Feedback
├─ Empfehlung an Klasse (freiwillig)
├─ Ggf. Testimonial für PR
└─ Weiterempfehlung an Kollegen
```

### Schule-für-Schule-Strategie

```
ZIEL: Kritische Masse pro Schule

Schule A:
├─ 1 Lehrer empfiehlt
├─ 20 Schüler nutzen D&D
├─ 5 Schüler werden KLAUSUR.RETTER Premium
├─ Word-of-Mouth in der Schule
├─ Weitere Lehrer werden aufmerksam
└─ Dominoeffekt

Dann: Nächste Schule
```

## Risiken & Mitigationen

| Risiko | Wahrscheinlichkeit | Mitigation |
|--------|-------------------|------------|
| PR zündet nicht | Mittel | Mehrere Angles testen, lokal starten |
| Lehrer haben keine Zeit | Hoch | Minimal-Aufwand, alles vorbereitet |
| Viralität bleibt aus | Hoch | Nicht darauf verlassen, Basis-Kanäle stärken |
| Social Media flopped | Mittel | Konsistenz > Perfektion, lange durchhalten |
